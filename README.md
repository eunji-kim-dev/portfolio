# MoodDot

감정 기록과 시각화를 중심으로 한 iOS 앱 프로젝트입니다.

## 프로젝트 개요
사용자가 하루의 감정 상태를 기록하고, 감정 데이터를 기반으로 자신의 상태를 돌아볼 수 있도록 설계한 앱입니다.

## 작업 기간
2025.11 ~ 2025.11

## 작업 인원
개인 프로젝트

## 사용 기술
Swift, SwiftUI, Codable, UserDefaults

## 주요 기능
- 색상 선택 기반 원터치 감정 기록 (기쁨·까칠함·슬픔·두려움·분노)
- 선택한 감정의 색상·피드백 문구 즉시 표시
- 날짜별 감정 기록 저장 및 조회 (Codable 기반 JSON 직렬화 → UserDefaults 저장)
- 월별 캘린더 시각화 및 기록 목록 표시

## 구현 특징
- 감정별 색상과 문구를 코드가 아닌 CSV 파일로 분리해, 로직 수정 없이 데이터 교체 가능
- 사용자 입력 → 상태 처리 → 저장·조회 → 화면 갱신까지의 전체 데이터 흐름을 직접 구현
