# MoodDot

감정 기록과 시각화를 중심으로 한 iOS 앱 프로젝트입니다.

## 프로젝트 개요
사용자가 하루의 감정 상태를 기록하고, 감정 데이터를 기반으로 자신의 상태를 돌아볼 수 있도록 설계한 앱입니다.

## 작업 기간
2025.06 ~ 2025.07

## 작업 인원
개인 프로젝트

## 사용 기술
Swift, iOS, Xcode

## 주요 기능
- 감정 기록 기능
- 날짜별 기록 확인
- 감정 상태 시각화
- 사용자 입력 데이터 저장 및 관리

## 담당 역할
- iOS 화면 구성
- 사용자 입력 처리
- 주요 기능 구현
- 발표자료 및 포트폴리오 정리
